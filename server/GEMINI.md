This file provides instructions and context for the Gemini large language model, specifically for the `server` directory.

# Backend Overview

The `server` directory contains the Node.js backend for the application. It is built with Express and uses tRPC for API endpoints.

# Key Directories

-   `src/routers`: Contains the tRPC routers for different parts of the application.
-   `src/db`: Contains the database schema and queries.
-   `src/_core`: Contains core functionality for the application.

# Running the Backend

To run the backend in development mode, run `pnpm dev` from the root directory.
