This file provides instructions and context for the Gemini large language model, specifically for the `client` directory.

# Frontend Overview

The `client` directory contains the React frontend for the application. It is built with Vite and uses TypeScript.

# Key Directories

-   `src/pages`: Contains the main pages of the application.
-   `src/components`: Contains reusable React components.
-   `src/lib`: Contains utility functions and the tRPC client.
-   `src/_core`: Contains core functionality for the application.

# Running the Frontend

To run the frontend in development mode, run `pnpm dev` from the root directory.
